# 汉字战场 · HanziCard

> 终端黑绿风格的中文汉字 PVE 卡牌游戏

---

## 快速开始

因为使用 ES Module，需要在 HTTP 服务器下运行（不能直接双击 `index.html`）。

```bash
# 方法 1：Python（最简单）
python3 -m http.server 8080
# 打开 http://localhost:8080

# 方法 2：Node.js npx
npx serve .

# 方法 3：VS Code Live Server 插件
# 右键 index.html → Open with Live Server
```

**GitHub Pages 部署**：直接推送整个文件夹到 `gh-pages` 分支，或在仓库 Settings → Pages 中选择 `main` 分支根目录，即可在线访问。

## 游戏玩法

| 操作 | 说明 |
|------|------|
| 点击手牌 | 直接出牌到场上（消耗能量） |
| 点击我方场牌 | 选中攻击者（绿色高亮） |
| 点击敌方场牌 | 对选中目标发动攻击 |
| 点击 AI 英雄图标 | 直接攻击敌方英雄（无嘲讽时） |
| 悬停任意卡牌 | 查看完整词条说明 |
| 结束回合 | AI 自动出牌+攻击 |

## 卡牌词条说明

| 词条 | 效果 |
|------|------|
| 横扫 | 攻击目标及其左右相邻卡牌（相邻卡受 60% 伤害） |
| 穿透 | 对场上卡造成伤害时，50% 同步直接伤害英雄 |
| 暴击 | 25% 概率造成双倍伤害 |
| 连击 | 15% 概率额外触发两次攻击（共 3 次） |
| 嘲讽 | 对方必须优先攻击嘲讽单位 |
| 护盾 | 出场时为己方英雄提供护盾（可格挡伤害） |
| 吸血 | 对目标造成伤害的 50% 回复自身生命 |
| 灼烧 | 击中后目标每回合损失 1 血，持续 3 回合 |
| 毒   | 击中后目标每回合损失 2 血，持续 3 回合 |
| 冻结 | 使目标跳过下次攻击 |
| 神圣护盾 | 吸收下一次任何伤害 |
| 护甲 | 每次受到伤害减少 1 点 |
| 狂怒 | 每消灭一个敌方卡牌，攻击力 +1 |
| 回响 | 出场时随机复制一张手牌 |
| 反制 | 打出时移除一张敌方卡牌的神圣护盾/护甲 |
| 枪出如龙 | 攻击目标同时对敌方所有其他卡牌造成 70% 伤害 |

## 花色合并

同一张卡（相同 id + 相同花色 + 相同数字）出现 **2 张或以上**在场上时，会自动合并，攻防数值直接叠加。

## 扩展开发

- **新增卡牌**：在 `data/cards.js` 的 `BASE_CARDS` 数组添加对象即可
- **新增词条效果**：在 `src/engine/GameState.js` 的 `_resolveAttack` 中添加对应分支
- **调整 AI**：修改 `GameState._aiTurn()` 的行动策略
- **样式调整**：所有颜色变量在 `assets/css/main.css` 的 `:root` 中统一管理
